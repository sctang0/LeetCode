```javascript
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var canJump = function(nums) {
    var lenPoint = nums.length - 1;
    var leftPos = lenPoint;
    for (var left = lenPoint; left >= 0; left--) {
        if (nums[left] + left >= leftPos) {
            leftPos = left;
        }
    }
    return leftPos == 0;
};
```



##### 复杂度分析

* 时间复杂度：O(n)，`n = nums.length;` 。遍历数组 nums 一遍。
* 空间复杂度：O(1)
